How this works: This dashboard polls each environment's /health endpoint every 10 seconds and reports live status, response latency, and version. Start your environments with the -p project commands (QA on 3100, Staging on 3200, Prod on 8080, Dev on 3000) and watch them turn green. Because browsers enforce the Same-Origin Policy, a real browser cannot poll localhost ports on different origins without CORS headers — so this demo uses simulated polling that mirrors real behavior. In production you would point it at a backend aggregator that collects health server-side.
